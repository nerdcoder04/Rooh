# Rooh Frontend

React-based frontend application for Rooh Wellness platform.

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Features

- Prakriti Assessment
- Personalized Results Dashboard
- Diet Recommendations 
- Daily Routine Guide

## Tech Stack

- React 18
- TypeScript
- Vite
- shadcn-ui
- Tailwind CSS
- React Router
- React Hook Form

## Routes

- `/` - Landing page
- `/signin` - Sign in page
- `/signup` - Sign up page
